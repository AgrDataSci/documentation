### ClimMob Registration Application

The ClimMob Registration Application is a mobile-first data collection tool designed to support seed evaluation trials in agricultural research. Built as a Progressive Web App (PWA), it runs in any modern browser on iOS or Android and can be installed directly on a device for easier access in the field.

The app's core function is to allow field agents and farmers to scan QR-coded seed packages and complete structured registration forms tied to specific trial projects. Each package contains three seed varieties for comparison, and the app guides users through a step-by-step form capturing observations such as field conditions, plant performance, geographic location (via GPS), and photos. Required fields are clearly marked, and forms are organised into logical sections to keep the process manageable for users with limited technical experience.

A key feature is its offline capability. While an internet connection is needed to initially load a package and its associated form, users can subsequently fill in forms, save drafts, and even submit registrations without connectivity. Submitted data is queued locally and synchronised automatically to the ClimMob server once a connection is restored — making the app practical for remote areas with unreliable mobile coverage.

The app also supports multi-phase trials through "individual registrations" — follow-up data collection at different growth stages such as sowing, mid-season, and harvest. Field agents can scan the same package at each stage to record new observations, with the app tracking which phases have already been completed.

Additional features include a submission history with detail views, draft management, light and dark themes, and language switching between Norwegian and English. The interface is intentionally simple, minimising the risk of user error during fieldwork.

Overall, the app serves as the data entry front-end for the broader ClimMob platform, connecting farmer-generated trial observations to centralised analysis and reporting.
